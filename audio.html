Lossy and Lossless Audio Compression


Lossless Audio Files

Examples of lossless audio file types

  • Windows Media Audio(WMA)
  • Audio Intechange File Format(AIFF)
  • MPEG-4 SLS
  • WAV
  • FLAC

Why use lossless audio file formats?

Lossless compression doesn't discard any data but still reduces the size of the file greatly.
In an audio file there is sound and silence. Lossless formats can compress the silence to nearly
nothing in terms of space while keeping everything else the same. This is important because
you know that you always have the same original music and that none of the sound quality is lost.


Lossy Audio Files

Examples of lossy audio file types

  • MP3
  • Vorbis
  • Windows Media Audio Lossy(WMA Lossy)
  • Opus

How does lossy compression work?

Lossy compression allows the file to be greatly reduced in size by removeing audio information and simplifying the data.
It works by removing some sounds or frequencies that aren't recognised as much and therfore don't affect the quality much.

Why use lossy audio compression?

Lossy compression is useful becuase it means that it takes up less space and is quicker to download or stream,
this is more preferable to use as nowadays people want everything done quickly.
